# Dynamic-Programming-for-Optimal-Metro-Station-Construction-Scheduling
The goal is to determine how many stations to build each year to minimize total discounted construction cost.

This project solves a metro station construction planning problem using backward induction dynamic programming over a 5-year horizon. Each year, urban growth creates demand for a certain number of new metro stations, but construction capacity, budget constraints, and a fixed mobilization cost make it suboptimal to build every station immediately as demand arises. The goal is to determine how many stations to build each year to minimize total discounted construction cost.
The state variable is the current backlog — the number of stations whose demand has materialized but have not yet been built. A maximum backlog of 1 is allowed, meaning demand can be deferred by at most one year. Each year, new demand (subdivisions) is added to the backlog, and the decision is how many stations to build. Construction costs vary by year (ranging from $1.0M to $1.8M per station), and a fixed mobilization cost of $2M is incurred whenever any construction takes place, regardless of how many stations are built. All costs are discounted at a factor of 0.9 per year, reflecting the time value of money.
The DP formulation defines f(t, s) as the minimum total discounted cost from year t onward given a backlog of s stations. The terminal condition requires the backlog to be fully cleared by the end of year 5 — any remaining backlog at year 6 is infeasible. Working backward, the algorithm evaluates every feasible number of stations to build at each year and state, checking that the resulting next-period backlog stays within the allowed range, and selects the option minimizing total cost.
The output traces the optimal construction schedule from year 1, showing backlog levels, stations built, yearly costs, and cumulative spending at each step. A full DP table is also printed, displaying optimal values and policies for all year-backlog combinations, allowing verification of the backward induction at every state.
